Over ons
Deze website is gemaakt door 2 studenten van het Media College in Amsterdam, voor de beroepsopdracht 'MuseumOnline'.

De opdracht
Voor deze opdracht moest er een website gamaakt worden met HTML en CSS. binnen deze website moest er gebruik gemaakt worden van de lessen van de afgelope piriode. Als laatste moets er gebruik gemaakt worden van WebXR om een VR element toe te voegen.

WebXR
WebXR is een script waarmee uw gemakkelijk en snel een VR (virtual reality) omgeving kunt toevoegen binnen uw website.
